This project focuses on analyzing a telecom customer churn dataset to identify the key factors influencing customer retention. Using Python libraries such as **Pandas, NumPy, Matplotlib, and Seaborn**, exploratory data analysis (EDA) was performed to visualize patterns and relationships between different services and churn behavior. The analysis shows that customers lacking services like online security, tech support, and device protection, as well as those using fiber optic internet, are more likely to churn. The main objective of this project is to generate meaningful insights that can help businesses improve customer satisfaction and reduce churn rates.
