# 📊 Telecom Customer Churn Analysis using Python

## 📌 Project Overview

Customer churn is one of the biggest challenges faced by telecom companies, as retaining existing customers is often more cost-effective than acquiring new ones. This project analyzes a telecom customer churn dataset to identify the key factors that influence customer retention and customer attrition.

Using Python and various data analysis libraries, Exploratory Data Analysis (EDA) was performed to uncover hidden patterns, trends, and relationships between customer demographics, subscribed services, and churn behavior. The insights generated from this project can help businesses make data-driven decisions to improve customer satisfaction, reduce churn rates, and increase long-term profitability.

---

- Identify the major factors contributing to customer churn.

## 📊 Key Insights

- Customers using **Fiber Optic Internet Service** have a higher churn rate.
- Customers without **Online Security** are more likely to leave.
- Lack of **Tech Support** significantly increases churn probability.
- Customers without **Device Protection** show higher churn.
- Month-to-month contract customers churn more frequently than long-term contract customers.
- Customers with shorter tenure have a higher probability of churn.
- Higher monthly charges are associated with increased churn.
- Customers using electronic check payment methods exhibit relatively higher churn.

---

## 💡 Business Recommendations

Based on the analysis, telecom companies should consider:

- Offering discounts for long-term contracts.
- Promoting Online Security and Tech Support services.
- Creating retention offers for new customers.
- Improving the quality of Fiber Optic Internet services.
- Introducing loyalty rewards for long-term customers.
- Providing personalized offers based on customer behavior.
- Monitoring high-risk customers through predictive analytics.
